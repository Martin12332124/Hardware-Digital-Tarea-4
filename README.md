# Tarea 4 - ISF-215 Hardware Digital
Módulo: ALU simple de 2 bits (AND, OR, XOR, suma)


Este circuito implementa una Unidad Aritmético Lógica (ALU) a nivel de compuertas lógicas discretas (chips 74HCxx). El módulo es capaz de procesar dos bits de entrada (A y B) y aplicarles una de cuatro operaciones posibles (AND, OR, XOR, o Suma Binaria con Acarreo), determinadas por el estado de dos bits selectores de control (S1 y S0).

Nota sobre diseño: El LED rojo representa el bit de resultado (Y), y el LED amarillo representa el bit de acarreo (Carry), activo exclusivamente en la suma binaria.
